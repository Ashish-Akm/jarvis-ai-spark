About J.A.R.V.I.S. AI

J.A.R.V.I.S. is a fully self-hosted, local-first AI Desktop Assistant that automates tasks, manages workflows, and integrates seamlessly with local OS APIs, hardware sensors, and cloud services (like Google Workspace) to increase your personal productivity.

Google API Limited Use Disclosure

Google API Disclosure: Jarvis AI's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

Key Features

  • Local-First: All models, local database config, and speech recognitions execute locally.
  • Workspace Automation: Fetch agendas, add calendar logs, coordinate tasks, and search emails directly via voice commands.
  • Hardware Telemetry: Live Mac/Windows sensors check (temperature, battery, lid monitoring).
