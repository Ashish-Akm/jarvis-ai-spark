Jarvis Assistant

Universal Cognitive OS & Desktop Assistant

System Overview

Jarvis Assistant is a fully self-hosted, local-first AI Desktop Assistant that automates tasks, manages workflows, and integrates seamlessly with local OS APIs, hardware sensors, and cloud services (like Google Workspace) to increase your personal productivity.

Google API Limited Use Disclosure

Google API Disclosure: Jarvis Assistant's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

Key Capabilities

  • LOCAL-FIRST ENGINE: All models, local database config, and speech recognitions execute locally.
  • WORKSPACE INTEGRATION: Fetch agendas, add calendar logs, coordinate tasks, and search emails directly via voice commands.
  • TELEMETRY & CONTROLS: Live Mac/Windows sensors check (temperature, battery, lid monitoring).
